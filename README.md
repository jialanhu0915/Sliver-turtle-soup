# Silver Turtle Soup 🍜

海龟汤（Situation Puzzle）— Claude Code 技能

## 这是什么

一个 Claude Code skill，让 AI 当汤主（出题者），你来提问和猜答案。

## 使用方式

1. 安装 Claude Code
2. 将本仓库 clone 到本地
3. 在 Claude Code 中说「开始游戏」即可开始

## 游戏规则

- AI 给出一个神秘情境
- 你只能问是/否问题
- AI 只回答：是 / 不是 / 是也不是 / 与此无关
- 猜出真相即为胜利

## 支持的风格

| 风格 | 说明 |
|------|------|
| 本格清汤 | 正统逻辑 + 温馨日常 |
| 本格红汤 | 正统逻辑 + 恐怖/血腥 |
| 变格清汤 | 变体风格 + 温馨日常 |
| 变格红汤 | 变体风格 + 恐怖/血腥 |
| 重口黑汤 | 重口味/黑暗系 |
| 整活抽象汤 | 整活/玩梗类 |

## 命令

| 命令 | 说明 |
|------|------|
| `开始游戏 [难度] [风格]` | 开始新游戏 |
| `提示` | 请求提示（每局最多3次） |
| `答案是：xxx` | 提交你的答案 |
| `认输` | 放弃并查看答案 |
| `退出` | 退出游戏 |

## 项目结构

```
SKILL.md                          # 技能定义（游戏规则）
database/
  templates/basic_templates.json  # 30道题目
  elements/                       # 元素库（场景/角色/物品/动作/结局）
generator/combine.py              # 题目生成器
```

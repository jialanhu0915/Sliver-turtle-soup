# Silver Turtle Soup 🍜

**通用 AI 海龟汤游戏技能包**

## 这是什么

一个适用于各类 AI 助手（Claude Code、ChatGPT、通义千问等）的海龟汤情景推理游戏技能。让 AI 扮演汤主，你来猜答案。

## 玩法

1. AI 给出神秘情境
2. 你只能问**是/否**问题
3. AI 回答：**是** / **不是** / **是也不是** / **与此无关**
4. 猜出真相即为胜利

## 命令

| 命令 | 说明 |
|------|------|
| `开始游戏` | 随机开始 |
| `开始游戏 [难度]` | 按难度（简单/中等/困难） |
| `开始游戏 [难度] [风格]` | 按难度+风格 |
| `提示` | 获取提示（每局最多3次） |
| `答案是：xxx` | 提交答案 |
| `认输` | 放弃，查看答案 |
| `退出` | 退出本局 |

## 难度

| 等级 | 说明 |
|------|------|
| 简单 | 线索多，逻辑短 |
| 中等 | 信息适中，有反转 |
| 困难 | 线索少，逻辑长 |

## 风格

| 风格 | 说明 |
|------|------|
| 本格清汤 | 正统逻辑 + 温馨日常 |
| 本格红汤 | 正统逻辑 + 恐怖/血腥 |
| 变格清汤 | 变体风格 + 温馨日常 |
| 变格红汤 | 变体风格 + 恐怖/血腥 |
| 重口黑汤 | 黑暗/重口味 |
| 整活抽象汤 | 玩梗/抽象 |

## 使用方法

将本仓库放入项目SKILLS目录即可。

## 项目结构

```
SKILL.md                       # 技能文件（完整游戏规则）
database/
  templates/                   # 30道题目模板
  elements/                    # 元素库（场景/角色/物品/动作/结局）
generator/combine.py           # 题目生成器
```
